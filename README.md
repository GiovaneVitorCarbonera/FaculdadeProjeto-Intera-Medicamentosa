# FaculdadeProjeto-Intera-oMedicamentosa
Projeto da Faculdade

este projeto é um sistema desenvolvido para auxiliar no controle e prevenção de interações medicamentosas, permitindo identificar riscos quando dois ou mais medicamentos são utilizados em conjunto.
Além disso, ele integra funcionalidades essenciais para o acompanhamento clínico, como o gerenciamento de medicamentos, pacientes e prescrições.

O sistema possibilita registrar medicamentos com suas características, cadastrar pacientes com informações relevantes e controlar prescrições contendo dosagens, horários e orientações médicas.
A partir desses dados, o sistema mantém um banco estruturado de interações medicamentosas, permitindo consultar substâncias envolvidas, nível de risco e detalhes de cada combinação potencialmente perigosa.

Dessa forma, o projeto oferece uma ferramenta completa para apoiar profissionais de saúde na segurança medicamentosa, garantindo mais qualidade e confiabilidade no tratamento dos pacientes.
